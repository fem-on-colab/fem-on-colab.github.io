    Packages¶

    You can install one of the packages provided by FEM on Colab by adding the following cell at the top of your notebook.

    FEniCS

    try: import dolfin except ImportError: !wget "https://fem-on-colab.github.io/releases/fenics-install.sh" -O "/tmp/fenics-install.sh" && bash "/tmp/fenics-install.sh" import dolfin
    • dolfin
    • mshr
    • multiphenics
    • RBniCS
